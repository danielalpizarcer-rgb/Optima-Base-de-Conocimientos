Instalación de OBM en Linux. 

1.- Instalar las dependencias faltantes. 
	dnf update
	sudo dnf install unzip nano perl libaio libXtst libxcrypt libxcrypt-compat libnsl openssl fontconfig m4
	sudo dnf install libpng<version>.x86_64    ->  sudo dnf install libpng12.x86_64 libpng12-devel.x86_64

	1.1 Comandos de isntalación sugeridos en la documentación
		
		- yum install -y unzip perl libaio libpng<version>.x86_64 libXtst libxcrypt libxcrypt-compat libnsl openssl fontconfig m4
	ó 
		- dnf install -y unzip perl libaio libpng<version>.x86_64 libXtst libxcrypt libxcrypt-compat libnsl openssl fontconfig m4
	
	1.2 configuraciones de sistema necesarias:

		- sudo sed -i '/net.ipv4.ip_local_port_range/d' /etc/sysctl.conf 
		  echo "net.ipv4.ip_local_port_range=31000 65535" | sudo tee -a /etc/sysctl.conf

		- sudo sysctl -p

	1.3 Limite de archivos abiertos
		
		- sudo nano -a /etc/security/limits.conf << 'EOF'
		  * hard nofile 30000
		  * soft nofile 30000
	 	  EOF

	1.4 Permisos de ejecución en /tmp

		- sudo mkdir -p /var/tmp/om_install
		- sudo chmod -R 777 /var/tmp/om_install
		- sudo mkdir -p /var/tmp/om_work
		- sudo chmod -R 777 /var/tmp/om_work
		- export IATEMPDIR=/var/tmp/om_install
		
		

	1.5 Todos los servidores deben estar sincronizados con la misma hora
		
		- timedatectl
		- sudo systemctl enable --now chronyd

	1.6 Descargar el paquete y descomprimirlo

	# unzip <paquete>
	# unzip OM_*_for_Linux.zip -d /var/tmp/om_install/fuentes
	# rpm -Ivh <paquete>

	1.7 Cerrar sesiópn SSH y volver a iniciar

1.- comprobar la entropia del sistema  -> versiones anteriores a 9.1 de RHEL 
		-> cat /proc/sys/kernel/random/entropy_avai
2.- Comprobar que /tmp/ no tiene : ovinstallparams.ini

3.- Crear un usuario para configuración de OBM que no sea adminitrador
	useradd 

	- sudo useradd -m omuser
	- echo "omuser soft nproc unlimited" | sudo tee /etc/security/limits.d/99-om-limits.conf

4.- Crear los archivos ini
	
	- cd ~
	- sudo nano ovinstallparams.ini
	
	Copiar las siguientes lines en el archivo

		Single Server
		
		[installer.properties]
		setup=HPOMi
		group=typical
	
	
		Gateway y Data Procesing
		
			Data processing server:

				[installer.properties]
				setup=HPOMi
				group=process

			Gateway server:

				[installer.properties]
				setup=HPOMi
				group=gateway


5.- Despliegue
	cd <carpeta de instalación>
	./install.sh
